# Setup Guides

This directory contains all setup and configuration guides for the Theglocal platform.

- **[Environment Setup](environment.md)** - Environment variables configuration and validation

## Quick Start

1. Start with [Environment Setup](environment.md) to configure all required environment variables
2. Follow [Supabase Setup](supabase.md) to set up your database
3. Review [Database Setup](database.md) for schema and migration information
4. Configure [MCP Server Setup](mcp-servers.md) for AI-powered operations
